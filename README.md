# 연습용 게시판

## 목표
- Spring Legacy Project를 이용하여 웹 구조에 대해 공부


## 사용기술

- Language: Java
- Database: Oracle 11gr2
- Server: Apache Tomcat 9
- Developer Tool: STS(Eclipse)
- Web: BootStrap, HTML, CSS, JavaScript, jQuery, Ajax, JSP, JSTL, EL
- Framework: Spring, MyBatis
